# Reports Domain

> Phase: Existing (basic) + Phase 7 (owner reports)
> Status: Basic PDF/CSV export built

---

## Overview

Reports generate structured outputs for property managers and property owners. Phase 7 introduces the owner statement — a monthly PDF report sent automatically to property owners via email.

---

## Existing Reports

- **Monthly P&L** — Revenue, expenses, net profit per property per month (PDF + CSV)
- **Multi-property summary** — All properties in one view (existing)
- **Expense category breakdown** — Pie chart + table
- **Airbnb CSV data export** — Raw Airbnb row data

### Auto-Distribution

Background job on 1st of month:
1. For each property → generate owner statement for prior month
2. For each property owner → send email with PDF attached
3. Log delivery in communication_logs
4. Make available in owner portal

## Report Service Interface

```typescript
interface ReportService {
  generateOwnerStatement(
    propertyId: string,
    periodMonth: Date
  ): Promise<OwnerStatement>;

  generatePDF(statement: OwnerStatement): Promise<Buffer>;

  sendOwnerStatementEmail(
    propertyId: string,
    periodMonth: Date,
    ownerEmail: string
  ): Promise<void>;

  exportMonthlyFinanceCSV(
    propertyId: string,
    periodMonth: Date
  ): Promise<string>;
}
```

## API Routes

| Method | Path | Description |
|--------|------|-------------|
| GET | /api/reports/owner-statement/:propertyId | Get/generate owner statement |
| GET | /api/reports/monthly-pdf/:propertyId | Download monthly P&L PDF |
| GET | /api/reports/activity/:propertyId | Activity log CSV |
| POST | /api/reports/send-owner-statement | Manually trigger email |
